# Customer Churn Prediction — Capstone Project

## Overview
This project focuses on predicting customer churn using machine learning techniques. The objective is to proactively identify customers at risk of leaving and enable targeted retention strategies.

---

## Business Problem
Customer churn represents a direct loss in revenue. Traditional retention efforts are reactive, often addressing churn only after it occurs. This project shifts from reactive to proactive retention using predictive analytics.

---

## Approach
- Data preprocessing and feature engineering  
- Exploratory Data Analysis (EDA)  
- Model development (Logistic Regression, Random Forest)  
- Model evaluation using accuracy, precision, recall, and ROC-AUC  

---

## Key Insights
- Customers within the first 6 months are at highest risk  
- Higher monthly charges correlate with churn  
- Month-to-month contracts show higher volatility
